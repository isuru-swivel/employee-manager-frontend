# employee-management frontend

## Instructions

##             * Clone the repository

##             * Create a file called .env.local in the root directory

##             * Add below line to .env.local file

```
NEXT_PUBLIC_API_URL=http://localhost:3001/api
```

##             * To install dependencies

```bash
npm install
```

##             * To run the app

```bash
npm run dev
```

##             * To run unit test

```bash
npm run test 
```

##             * Storybook

```bash
npm run storybook
```
